# 面试记忆本

一款本地优先的面试准备工具，帮助你把分散的项目资料、面试问答和复盘记录，整理成可持续练习的个人题库。

## 它能做什么

- 按项目管理面试题库
- 支持手动录入、批量导入、本地智能映射导入
- 支持闪卡练习、出声练习、每日计划和统计复盘
- 支持 AI 工作台：模拟面试、整理问答草稿、导出 Markdown、确认后导入题库
- 支持用户自定义模型配置与提示词

## 适合谁

- 正在准备产品经理 / AI 产品经理面试的人
- 想把项目经历系统化整理成问答的人
- 希望通过主动回忆和间隔复习提升面试表达的人

## 使用方式

这是一个纯前端静态页面，打开 `index.html` 即可使用。

### 本地打开

直接在浏览器中打开仓库里的 `index.html`。

### 在线访问

如果你将本仓库开启 GitHub Pages，也可以通过公开网址直接访问。

## 数据说明

- 题库数据默认保存在浏览器本地 `localStorage`
- 不同浏览器、不同设备之间不会自动同步
- 清理浏览器站点数据后，本地内容可能会被清空
- 建议定期导出 JSON 备份

## AI 功能说明

- AI 工作台支持用户手动配置 `Base URL / API Key / Model`
- 支持单模型、双模型、混合模式模拟面试
- 支持导出 Markdown，或确认后导入题库
- 当前更适合个人本地使用

## 隐私与分享

这个公开仓库版本为干净分享版，不包含个人题库数据。

如果你想分享给朋友，建议通过：

1. 仓库地址直接分享
2. 开启 GitHub Pages 后分享公开网址
